Logout Form

  • Tombol logout menyatu dengan dashboard.
  • Wajib ada konfirmasi untuk logout

Logout Process

Pre-kondisi:
  • User harus sedang login, jika state sedang tidak login maka tombol logout tersembunyi
After-logout:
  • Terdapat pesan Anda sudah logout
  • Terdapat timer 2 detik untuk pesan diatas kemudian otomatis redirecting ke Form Login

Simple Register Form

Input user :
  • Nama User:
    • Required
    • Length: 3 - 20 char
  • Username:
    • Required
    • Length: 3 - 20 char
  • Password:
    • Required
    • Length: 6 - 20 char
  • Konfirmasi Password:
    • Required
    • Length: 6 - 20 char

Simple Register Process

Cek Duplikasi Username:
  • Jika Duplikat tampilkan pesan error: Maaf, username [username] telah diambil oleh orang lain. Silahkan pakai username lainnya!
  • Jika tidak, tampilkan pesan sukses: Creating username accepted.
Enkripsi password:
  • Enkripsi password dengan md5 sebelum INSERT ke tabel user

Advanced Register Form

Input user :
  • Nama User, Client Side Validation with JS:
    • Uppercase only!
    • Tanpa spasi
    • Tanpa special character
    • Tanpa angka [0-9]
  • Username, Client Side Validation with JS:
    • Lowercase only!
    • Tanpa spasi
    • Tanpa special character
  • Password:
    • Hints for Autofill Strength Random Password (password acak default untuk user)
    • Jika user menghiraukan hints maka Validasi Konfirmasi Password harus ada, jika meng-klik hints maka password dan konfirmasinya otomatis terisi dengan autofill
  • Konfirmasi Password, Client Side Validation with JS:
    • Wajib identik dengan Password
    • Matikan tombol submit jika tidak identik

Advanced Register Process

Validasi dilakukan kembali di sisi server by PHP untuk mengecek:
  • Validitas Nama User
  • Validitas Username
  • Password Strength

Simple Dashboard

Saat state login:
  • Munculkan teks: Welcome [nama user] Anda login sebagai [hak-akses-user]
Saat tidak login atau timeout session (30 menit web ditinggalkan user):
  • Munculkan teks: Session login Anda habis. Silahkan relogin!
  • Terdapat timer 2 detik untuk pesan diatas kemudian otomatis redirecting ke Form Login
